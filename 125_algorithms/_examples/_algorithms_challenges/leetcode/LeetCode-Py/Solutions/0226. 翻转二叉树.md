# [0226. 翻转二叉树](https://leetcode.cn/problems/invert-binary-tree/)

- 标签：树、递归
- 难度：简单

## 题目大意

给定一个二叉树，将其进行左右翻转。

## 解题思路

从根节点开始遍历，然后从叶子节点向上递归交换左右子树位置。

## 代码

```Python
class Solution:
    def invertTree(self, root: TreeNode) -> TreeNode:
        if root == None:
            return root
        left = self.invertTree(root.left)
        right = self.invertTree(root.right)
        root.left = right
        root.right = left
        return root
```
